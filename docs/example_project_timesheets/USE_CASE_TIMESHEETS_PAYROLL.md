# Caso de Uso Demo - Timesheets y Payroll

Quiero un sistema web para una empresa pequena donde:

- un administrador registre empleados y su tarifa por hora;
- cada empleado registre inicio y fin de jornada;
- cada empleado cargue tareas realizadas y horas trabajadas;
- un supervisor apruebe o rechace horas;
- el sistema calcule pago semanal por empleado;
- se pueda exportar un reporte CSV;
- exista historial auditable de cambios y aprobaciones.

Condiciones iniciales:

- preferimos Python para backend y automatizacion;
- la primera entrega puede ser web-only;
- el sistema debe poder operar con pocos usuarios, pero dejando base para crecer;
- no queremos inventar integraciones externas no mencionadas.
